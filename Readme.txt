You must check pre-requirements to execute this project on you machine:
1- download or clone de project on you machine
2- Node must be installed
3- yarn must be installed
4- install dependencies of this project typing "yarn" inside the folder of this project
5- type yarn dev on a terminal to run the project. The browser will open a new tab showing the interface. Just choose a fund and you will see data about fund. 

NOTE: The data about funds were randomic generated. The objective isn't show fund data, but I just intended put React in practice in order to learn about it.
